Aurdino UNO

The Arduino Uno serves as the brain of

our dustbin. It controls and coordinates.

all the components in the system.

Odour Sensor

The odour sensor continuously monitors the air

around the dustbin, detecting a significant increase

in unpleasant odors and sending this data to the

Arduino uno.
Ultrasonic Sensor
Ultrasonic sensors: the ultrasonic sensor utilizes

soundwaves to measure the distance between the

dustbin lid and the waste inside. It helps determine

the how full the dustbin is

Wi-Fi Module
The wi-fi module connects the smart dustbin to

a network , allowing it to transmit data wirelessly.

This data can be sent to a remote server or cloud

platform for further analysis
